# Домино — выкладывание в ряд

Консольная программа на C++ для работы с набором костяшек домино. Позволяет вручную управлять набором костяшек и проверять, можно ли выложить их в непрерывную цепочку.

---

## Возможности

- Добавление костяшек вручную или из файла
- Удаление и изменение костяшек в наборе
- Вывод текущего набора
- Проверка, можно ли выложить все костяшки в ряд (с выводом цепочки)

---

## Требования

- **Компилятор**: C++26
- **CMake**: версия 4.0 и выше

---

Исполняемый файл появится в папке `build/` под именем `prod`.

---

После запуска отображается меню:

```
1. Добавить костяшку
2. Удалить костяшку
3. Изменить костяшку
4. Загрузить из файла data.txt
5. Вывести костяшки
6. Проверить, можно ли выложить в ряд
Выбор:
```

---

## Формат данных

Каждая костяшка — строка из **двух цифр**, например `35` означает домино, содержащее с одной половины 3, с другой 5.

### Ввод вручную (пункт 1)

```
Введите значение: 35
Добавлено 35
```

### Загрузка из файла (пункт 4)

Поместите файл `data.txt` рядом с исполняемым файлом. Каждое значение — через пробел или на новой строке:

```
11 23 31
```

Невалидные значения автоматически пропускаются с пояснением.

---

## Проверка цепочки (пункт 6)

Программа использует **поиск с возвратом (backtracking)**: перебирает все возможные порядки и расположения костяшек, пока не найдёт цепочку, в которой правый конец каждой костяшки совпадает с левым концом следующей.

**Пример:**

Набор: `12 21 13`

```
можно: 31, 12, 21
```

Если цепочка невозможна:

```
Нельзя
```

## Валидация

`CorrectDomino::Check` отклоняет костяшку, если:
- длина строки не равна 2,
- в строке есть нецифровые символы.

`CorrectNumber` гарантирует корректный ввод целого числа по определённым правилам, бесконечно переспрашивает при ошибке.
